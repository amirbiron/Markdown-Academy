# הצעות לשיפורים ופיצ'רים חדשים - Markdown Academy

## סוגי תכנים חדשים (שיעורים נוספים)

### 1. שיעור: Math Equations עם LaTeX/KaTeX
- **תיאור**: לימוד כתיבת נוסחאות מתמטיות בתוך Markdown באמצעות תחביר LaTeX
- **תוכן**:
  - נוסחאות inline: `$E = mc^2$`
  - בלוקי נוסחאות: `$$\sum_{i=1}^{n} x_i$$`
  - שברים, חזקות, שורשים ואינטגרלים
- **קהל יעד**: סטודנטים, חוקרים, מתכנתים שכותבים דוקומנטציה טכנית
- **רמה**: מתקדם
- **שינויים טכניים**: הוספת ספריית `remark-math` ו-`rehype-katex` לפרויקט

### 2. שיעור: Markdown ל-README מקצועי
- **תיאור**: איך לבנות README מושלם לפרויקט ב-GitHub
- **תוכן**:
  - מבנה README סטנדרטי (תיאור, התקנה, שימוש, תרומה, רישיון)
  - Badges (תגיות סטטוס) - build status, coverage, version
  - שילוב תמונות ו-GIFs להדגמה
  - טבלת תוכן עניינים (Table of Contents)
  - Shields.io badges
- **רמה**: בינוני
- **ערך מוסף**: כל מפתח צריך לדעת לכתוב README טוב

### 3. שיעור: Definition Lists ו-Abbreviations
- **תיאור**: רשימות הגדרות וקיצורים - תכונות Markdown פחות מוכרות
- **תוכן**:
  - רשימות הגדרות (מונח + הסבר)
  - קיצורים שמוצגים ב-tooltip
  - שימושים מעשיים בדוקומנטציה טכנית
- **רמה**: מתקדם

### 4. שיעור: Emoji ב-Markdown
- **תיאור**: שימוש באמוג'י ב-Markdown עם shortcodes
- **תוכן**:
  - תחביר emoji shortcodes: `:rocket:` → 🚀
  - אמוג'ים נפוצים בהקשר טכני
  - שימוש באמוג'י ב-commit messages ו-PR descriptions
  - Conventional Commits עם אמוג'ים (gitmoji)
- **רמה**: יסודות

### 5. שיעור: Front Matter (YAML/TOML)
- **תיאור**: מטא-דאטה בראש קבצי Markdown
- **תוכן**:
  - תחביר YAML front matter (`---`)
  - שדות נפוצים: title, date, author, tags
  - שימושים ב-Jekyll, Hugo, Next.js, Docusaurus
  - TOML front matter (`+++`)
- **רמה**: מתקדם

### 6. שיעור: Admonitions / Callouts
- **תיאור**: תיבות הדגשה (note, warning, tip, danger)
- **תוכן**:
  - תחביר GitHub: `> [!NOTE]`, `> [!WARNING]`, `> [!TIP]`
  - שימושים בדוקומנטציה
  - עיצוב callouts מותאם אישית
- **רמה**: בינוני
- **שינויים טכניים**: תמיכה ב-GitHub-style alerts ב-renderer

## פיצ'רים חדשים לפלטפורמה

### 7. מצב אתגר (Challenge Mode)
- **תיאור**: אתגרי Markdown מתוזמנים שבודקים מהירות ודיוק
- **איך זה עובד**:
  - המשתמש מקבל תמונה של תוצאה רצויה
  - צריך לשחזר אותה ב-Markdown תוך זמן מוגבל
  - ניקוד על סמך דיוק + מהירות
- **ערך מוסף**: גיימיפיקציה עמוקה יותר, תחרותיות

### 8. Cheat Sheet אינטראקטיבי
- **תיאור**: דף עזר מהיר עם כל תחביר Markdown במקום אחד
- **פיצ'רים**:
  - חיפוש מהיר
  - קטגוריות מתקפלות
  - כפתור "נסה עכשיו" שפותח את ה-sandbox עם הדוגמה
  - אפשרות העתקה מהירה

### 9. ייצוא לפורמטים שונים
- **תיאור**: אפשרות לייצא את ה-Markdown מה-sandbox לפורמטים שונים
- **פורמטים**:
  - HTML
  - PDF
  - Word (docx)
  - קובץ `.md` להורדה
- **שינויים טכניים**: שימוש בספריות כמו `html-pdf` או `jspdf`

### 10. מערכת Markdown Templates
- **תיאור**: תבניות מוכנות מראש שאפשר לטעון ל-sandbox
- **תבניות לדוגמה**:
  - README לפרויקט
  - מסמך API
  - רשומת בלוג
  - CV/קורות חיים
  - Changelog
  - Meeting notes
  - Bug report
- **איך זה עובד**: כפתור "בחר תבנית" ב-sandbox שטוען תוכן ראשוני

### 11. לוח הישגים ודירוג (Leaderboard)
- **תיאור**: טבלת דירוג בין משתמשים
- **קריטריונים**:
  - מספר שיעורים שהושלמו
  - ציוני אתגרים
  - רצף ימים רצופים (streak)
  - הישגים מיוחדים
- **שינויים טכניים**: טבלת DB חדשה לניקוד, API endpoints חדשים

### 12. מצב תרגול Diff
- **תיאור**: הצגת ההבדלים בין הפתרון של המשתמש לפתרון המומלץ
- **איך זה עובד**:
  - המשתמש כותב Markdown
  - לוחץ "בדוק"
  - מקבל diff חזותי צבעוני (ירוק/אדום) שמראה מה חסר ומה שונה
- **שינויים טכניים**: ספריית diff כמו `diff` או `jsdiff`

### 13. שיתוף קטעי Markdown (Snippets)
- **תיאור**: אפשרות לשמור ולשתף קטעי Markdown
- **פיצ'רים**:
  - שמירת snippets אישיים
  - יצירת קישור שיתוף ייחודי
  - גלריית snippets ציבוריים מהקהילה
  - אפשרות fork של snippet של אחרים
- **שינויים טכניים**: טבלת snippets חדשה ב-DB, endpoints לשיתוף

### 14. Markdown Linter מובנה
- **תיאור**: בדיקת שגיאות ותיקון אוטומטי ב-Markdown
- **פיצ'רים**:
  - סימון שגיאות תחביר בזמן אמת
  - הצעות לתיקון
  - כללי style (למשל: כותרות לא מדורגות, שורות ארוכות מדי)
  - אינטגרציה עם CodeMirror כ-extension
- **שינויים טכניים**: שימוש ב-`markdownlint` כ-library

### 15. Daily Markdown Tip
- **תיאור**: טיפ יומי שמוצג בדף הבית
- **פיצ'רים**:
  - טיפים מתחלפים כל יום
  - אפשרות "נסה עכשיו" שפותח את ה-sandbox
  - ארכיון של טיפים קודמים
  - התראות push אופציונליות
